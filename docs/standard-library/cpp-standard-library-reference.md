# <a name="c-standard-library-reference"></a>C++ 표준 라이브러리 참조

C++ 프로그램은 C++ 표준 라이브러리에 따른 이러한 구현에서 여러 함수를 호출할 수 있습니다. 이러한 함수는 입력 및 출력과 같은 핵심 서비스를 수행하고 자주 사용되는 작업의 효율적인 구현을 제공합니다.
