# <a name="tznamemax"></a>TZNAME_MAX

**사용되지 않습니다**. 표준 시간대 이름 변수의 허용 가능한 최대 문자열 길이입니다. 이 매크로는 Visual Studio 2012 및 이전 버전의 \<limits.h>에 정의되어 있습니다. Visual Studio 2013 및 이후 버전에서는 정의되어 있지 않습니다. 현재 표준 시간대 이름을 포함하는 데 필요한 길이를 얻으려면 [_get_tzname](../c-runtime-library/reference/get-tzname.md)을 사용합니다.

## <a name="syntax"></a>구문

```
#include <limits.h>
```
