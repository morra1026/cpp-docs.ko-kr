# <a name="compiler-error-c3632"></a>컴파일러 오류 C3632

'event': 구문에 대 한 이벤트 스타일이 잘못 되었습니다

[__event](../../cpp/event.md) 선언은 모든 구문에서 잘못 된 것입니다.

C3632 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
