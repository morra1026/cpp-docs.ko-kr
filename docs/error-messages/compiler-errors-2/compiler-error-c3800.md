# <a name="compiler-error-c3800"></a>컴파일러 오류 C3800

'declaration': 속성 및 이벤트를 혼합할 수 없습니다

구문 속성 및 이벤트를 선언할 수 없습니다.

C3800 사용 되지 않는 컴파일러 옵션을 사용 하 여 전용인 **/clr: oldsyntax**합니다.
