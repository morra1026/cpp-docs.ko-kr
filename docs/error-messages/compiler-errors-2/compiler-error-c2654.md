# <a name="compiler-error-c2654"></a>컴파일러 오류 C2654

'identifier': 멤버 함수 외부에서 멤버 액세스 시도

멤버 선언에서 액세스 됩니다. 멤버 데이터는 멤버 함수에만 액세스할 수 있습니다.

이 오류는 선언에서 변수를 초기화 하려고 할 때 발생할 수 있습니다. 이 목적을 위해 생성자를 사용 합니다.
