# <a name="nmake-fatal-error-u1052"></a>NMAKE 심각한 오류 U1052

> 파일 '*filename*' 찾을 수 없음

NMAKE는 다음 중 하나를 사용 하 여 지정 된 파일을 찾을 수 없음:

- **/F** 옵션

- **! 포함** 전처리 지시문

- At 기호 (**\@**) 지시 파일 지정자

파일이 파일 이름 철자가 올바른지 확인 합니다.
