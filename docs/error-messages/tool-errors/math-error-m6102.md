# <a name="math-error-m6102"></a>수학 오류 M6102

비정상

작업 생성으로 인해 유효 하지 않게 하는 매우 작은 부동 소수점 숫자를 significance의 손실입니다. Denormal 부동 소수점 예외는 일반적으로 마스크 트래핑 되 고 작업할 수 있습니다.

프로그램은 130 종료 코드로 종료 됩니다.
