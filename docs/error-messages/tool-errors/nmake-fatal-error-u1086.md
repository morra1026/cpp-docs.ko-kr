# <a name="nmake-fatal-error-u1086"></a>NMAKE 심각한 오류 U1086

**유추 규칙 종속 파일을 사용할 수 없습니다.**

콜론 (**:**)을 유추 규칙 뒤에 야 다음 중 하나에서:

- 줄 바꿈 문자

- 세미콜론 (**;**), 명령에 올 수 있습니다.는

- 숫자 기호 (**#**), 메모에 올 수 있습니다는
