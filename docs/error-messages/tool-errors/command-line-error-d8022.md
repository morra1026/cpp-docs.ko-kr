# <a name="command-line-error-d8022"></a>명령줄 오류 D8022

'messagefile'을 열 수 없습니다.

지정된 된 파일 경로 환경 변수에 지정 된 디렉터리 또는 현재 디렉터리에 없는 경우 컴파일러 명령줄 구문 및 옵션의 간략 한 요약을 포함 하는 메시지 파일입니다.

현재 디렉터리 또는 현재 경로에 디렉터리를이 파일을 이동 합니다. 이 파일을 찾을 수 없는 경우 설치 디스크에서 복사 하려면 설치 프로그램을 실행 합니다.
