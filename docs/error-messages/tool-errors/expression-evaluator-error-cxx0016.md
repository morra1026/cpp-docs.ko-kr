# <a name="expression-evaluator-error-cxx0016"></a>식 계산기 오류 CXX0016

상수가 너무 큽니다.

C 식 계산기는 4,294,967,295 (16 진수 0FFFFFFFF) 보다 큰 부호 없는 정수 상수 또는 해당 크기가 약 1.8 e + 308 보다 큽니다. 부동 소수점 상수에 수락할 수 없습니다.

이 오류는 can0016과 동일 합니다.
