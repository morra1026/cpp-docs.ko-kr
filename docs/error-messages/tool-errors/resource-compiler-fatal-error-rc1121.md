# <a name="resource-compiler-fatal-error-rc1121"></a>리소스 컴파일러 심각한 오류 RC1121

파일을 읽는 동안 I/O 오류가 발생했습니다.

리소스 컴파일러가 파일을 읽을 수 없습니다. 파일이 포함된 드라이브를 사용할 수 있고 파일이 유효한지 확인합니다.
