# <a name="expression-evaluator-error-cxx0041"></a>식 계산기 오류 CXX0041

클래스 요소는 정적 멤버 또는 멤버 함수 여야 합니다.

클래스 (또는 구조체나 공용 구조체)의 비정적 멤버는 클래스의 인스턴스화를 지정 하지 않고 사용 하는 데 사용 되었습니다.

인스턴스화를 지정 하지 않고 정적 데이터 멤버 또는 멤버 함수를 사용할 수 있습니다.

이 오류는 can0041과 동일 합니다.
