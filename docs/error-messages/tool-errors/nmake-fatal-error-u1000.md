왼쪽된 괄호 **(** 는 일치 하는 오른쪽 괄호 없이 표시, **)** 를 매크로 호출 합니다. 올바른 형식은 **$(**<em>이름</em>**)**; **$** <em>n</em> 한 문자로 된 이름을 허용 됩니다.
