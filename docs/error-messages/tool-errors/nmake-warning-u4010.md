# <a name="nmake-warning-u4010"></a>NMAKE 경고 U4010

'target': 빌드하지 못했습니다. /K 지정을 계속 하는 중...

지정된 된 대상에 대 한 명령 블록의 명령은 0이 아닌 종료 코드를 반환 합니다. /K 옵션 NMAKE 빌드 관련 되지 않은 부분을 계속 처리 및 NMAKE 세션이 완료 되 면 종료 코드 1을 발급 하도록 지시 합니다.

NMAKE 경고 이면 지정된 된 대상 자체를 다른 대상에 종속 [U4011](../../error-messages/tool-errors/nmake-warning-u4011.md) 이 경고 다음입니다.
