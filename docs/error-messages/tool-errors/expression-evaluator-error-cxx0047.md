# <a name="expression-evaluator-error-cxx0047"></a>식 계산기 오류 CXX0047

인수 목록이 함수를 일치 하지 않습니다.

프로그램에 정의 된 동일한 이름의 모든 함수를 형식 매개 변수 목록과 일치 하지 않는 실제 매개 변수 목록 가진 함수를 호출 하는 식입니다.

일치 하는 정확한 매개 변수 또는 개체를 생성 하지 않아도 되는 일치 하는 경우에 오버 로드 된 함수를 호출할 수 있습니다.

이 오류는 can0047과 동일 합니다.
