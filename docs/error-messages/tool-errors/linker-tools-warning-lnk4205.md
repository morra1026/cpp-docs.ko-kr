# <a name="linker-tools-warning-lnk4205"></a>링커 도구 경고 LNK4205

'filename'에 참조 모듈에 대 한 현재 디버깅 정보가 없습니다. 디버그 정보가 없는 것 처럼 개체를 링크 합니다.

.Pdb 파일에 오래 된 정보입니다. 링커가는 디버그 정보 없이 개체를 연결할 계속 합니다. 사용 하 여 개체 파일을 다시 컴파일할 수도 있습니다는 [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) 옵션입니다.
