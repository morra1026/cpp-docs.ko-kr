# <a name="compiler-warning-level-1-c4631"></a>컴파일러 경고(수준 1) C4631

MSXML 또는 XPath를 사용할 수 없습니다. XML 문서 주석이 처리되지 않습니다. 이유

공용 언어 런타임 설치에 문서 주석의 처리를 지원하는 데 필요한 파일이 없습니다. 공용 언어 런타임 다시 설치
