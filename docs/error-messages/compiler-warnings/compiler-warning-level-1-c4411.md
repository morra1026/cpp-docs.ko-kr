# <a name="compiler-warning-level-1-c4411"></a>컴파일러 경고(수준 1) C4411

'identifier': 치환 레지스터가 되는 기호

식별자는 로컬 기호 치환 레지스터가 되는 다른 기호를 사용 하 여 피연산자에 사용할 수 있습니다.
