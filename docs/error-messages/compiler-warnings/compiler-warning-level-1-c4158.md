# <a name="compiler-warning-level-1-c4158"></a>컴파일러 경고(수준 1) C4158

A **#pragma pointers_to_members (** *single* &#124; *다중* &#124; *virtual* **)** 발생 없이 실행 되었습니다 **#pragma pointers_to_members (full_generality)** 합니다.
