# <a name="compiler-warning-level-3-c4066"></a>컴파일러 경고(수준 3) C4066

와이드 문자 상수에서 첫 문자 뒤의 문자는 무시됩니다.

컴파일러는 와이드 문자 상수의 첫 문자만 처리합니다.
