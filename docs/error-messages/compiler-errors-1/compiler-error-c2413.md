# <a name="compiler-error-c2413"></a>컴파일러 오류 C2413

'token': 맞춤 크기가 잘못 되었습니다

사용 하 여 사용 되는 크기는 `ALIGN` 지시문이 없습니다. 또는 유효한 범위를 벗어납니다.
