# <a name="compiler-error-c2047"></a>컴파일러 오류 C2047

기본값이 잘못되었습니다.

`default` 키워드는 `switch` 문에만 나타날 수 있습니다.

다음 샘플에서는 C2047을 생성합니다.

```
// C2047.cpp
int main() {
   int i = 0;
   default:   // C2047
   switch(i) {
      case 0:
      break;
   }
}
```

해결 방법:

```
// C2047b.cpp
int main() {
   int i = 0;
   switch(i) {
      case 0:
      break;
      default:
      break;
   }
}
```
