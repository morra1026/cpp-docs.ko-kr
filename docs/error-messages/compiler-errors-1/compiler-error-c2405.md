# <a name="compiler-error-c2405"></a>컴파일러 오류 C2405

오프셋이 있는 간단한 전방 참조가 잘못 되었습니다.

간단한 전방 참조만 레이블을 참조 해야 합니다. 추가 오프셋을 사용할 수 없습니다.
