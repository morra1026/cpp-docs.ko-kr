# <a name="fatal-error-c999"></a>심각한 오류 C999

Visual c + + 도움말 메뉴에서 [기술 지원] 명령을 선택 또는 자세한 정보에 대 한 기술 지원 도움말 파일을 열고 알 수 없는 메시지.

이 오류는 보통 여러 컴파일러 버전으로 파일이 혼합되어 있거나 설치된 컴파일러가 손상되었음을 의미합니다. 사용 된 **프로그램 및 기능** 을 복구 하거나 제품을 다시 설치 하려면 제어판에 애플릿을 합니다.
