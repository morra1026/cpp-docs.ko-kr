# <a name="fatal-error-c1051"></a>심각한 오류 C1051

프로그램 데이터베이스 파일에 'pdbfile'에 삭제 하 고 다시 컴파일해야 사용 되지 않는 형식이 있습니다.

컴파일러는 이전 버전 번호는 프로그램 데이터베이스 파일을 업데이트할 수 없습니다. 파일을 삭제 하 고 사용 하 여 프로그램을 다시 컴파일해야 **/Zi** 하거나 **/ZI**합니다. 자세한 내용은 참조 하세요.  [ /z7, /Zi, /ZI (디버그 정보 형식)](../../build/reference/z7-zi-zi-debug-information-format.md)
