# <a name="catalog-information--mfc-data-access"></a>카탈로그 정보  (MFC Data Access)

데이터 소스의 테이블에 대한 정보에는 테이블 및 테이블 내 열의 이름, 테이블 권한, 기본 키와 외래 키의 이름, 미리 정의된 쿼리나 저장 프로시저에 대한 정보, 테이블의 인덱스 관련 정보, 그리고 테이블 관련 통계가 포함될 수 있습니다.

자세한 내용은 [데이터 원본: 데이터 원본 (ODBC)의 스키마 확인](../data/odbc/data-source-determining-the-schema-of-the-data-source-odbc.md)합니다.
