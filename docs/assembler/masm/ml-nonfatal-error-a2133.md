# <a name="ml-nonfatal-error-a2133"></a>ML 심각하지 않은 오류 A2133

**등록 호출을 덮어써 값**

레지스터 프로시저에 인수로 전달 되었지만에서 생성 된 코드 [INVOKE](../../assembler/masm/invoke.md) 다른 인수를 전달 하는 레지스터의 내용을 제거 합니다.

AX, AL, AH, EAX, DX, DL, DH, 및 EDX 레지스터 데이터 변환을 수행 하는 어셈블러에서 사용할 수 있습니다.

다른 레지스터를 사용 합니다.
