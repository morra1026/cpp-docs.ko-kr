# <a name="ml-nonfatal-error-a2078"></a>ML 심각하지 않은 오류 A2078

**명령에 극동 없도록 간접 주소 지정**

조건부 이동 또는 루프 메모리 피연산자를 사용할 수 없습니다. 상대 주소 또는 레이블 지정 되어야 합니다.
