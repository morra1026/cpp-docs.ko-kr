# <a name="invoke"></a>INVOKE

제공한 주소로 프로시저 호출 *식*, 언어 형식의 표준 호출 규칙에 따라 레지스터 나 스택에 인수를 전달 합니다.

## <a name="syntax"></a>구문

> INVOKE *식을* [[합니다 *인수*]]

## <a name="remarks"></a>설명

프로시저에 전달 되는 각 인수 식을, 레지스터 쌍으로, 또는 주소 식일 수 있습니다 (앞에 식을 `ADDR`).
