# <a name="real4"></a>REAL4

할당 하 고 필요에 따라 각각에 대 한 단 정밀도 (4 바이트) 부동 소수점 숫자를 초기화 *이니셜라이저*합니다.

## <a name="syntax"></a>구문

> 이름 REAL4 이니셜라이저 [[, 이니셜라이저]]...
