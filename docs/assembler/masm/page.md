# <a name="page"></a>PAGE

줄을 설정 하는 첫 번째 지시문 *길이* 문자 *너비* 프로그램 목록입니다. 인수 없이 지정 된 경우 페이지 나누기를 생성 합니다. 두 번째 지시문 섹션 번호를 증가 및 페이지 번호를 1로 다시 설정 합니다.

## <a name="syntax"></a>구문

> 페이지 [[*길이*]] [[합니다 *너비*]]<br/><br/>
> 페이지 +
