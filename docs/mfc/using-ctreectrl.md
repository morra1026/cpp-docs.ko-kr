# <a name="using-ctreectrl"></a>CTreeCtrl 사용

트리 컨트롤을 클래스에 의해 표현 [CTreeCtrl](../mfc/reference/ctreectrl-class.md)를 디스크에 인덱스 또는 파일 및 디렉터리의 항목 머리글 문서 등의 항목의 계층적 목록을 표시 하는 창입니다. 각 항목 레이블 및 선택적 비트맵 이미지로 구성 되며 각 항목에 연결 된 하위 항목의 목록을 가질 수 있습니다. 항목을 클릭 하 여 사용자를 확장 하 고 연결 된 목록을 하위 항목 축소 수 있습니다. 파일 탐색기의 왼쪽 창에서 디렉터리 트리는 트리 컨트롤의 예시입니다.
